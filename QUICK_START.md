# ⚡ Quick Start Guide

## 🎯 One-Minute Setup

### Step 1: Access the Games
```
Visit: https://School-projects1234.github.io/Sparxscience
```

### Step 2: Find the Secret
- Look at the **bottom right corner** of the page
- **Double-click** the small area (it won't be visible!)

### Step 3: Solve the CAPTCHA
- Read the distorted code in the box
- Type it in the input field
- Click "Verify"

### Step 4: Enter Passcode
- The secret code is: **PHOENIX**
- Type it in the password field
- Click "Submit"

### Step 5: Choose Your Game
- Select from War Thunder gamemodes (Aerial, Tank, Survival, etc.)
- Or try other games (coming soon)

### Step 6: Play!
- **WASD or Arrows** - Move your vehicle
- **Mouse** - Look around and aim
- **Click or Space** - Fire weapons
- **Shift** - Descend (air) / Reverse (ground)
- **Space** - Ascend (air) / Accelerate (ground)

## 🕵️ Pro Tips

1. **Can't find the trigger?**
   - It's a 30x30px invisible box in the bottom right corner
   - Or use keyboard shortcut: **Ctrl+Shift+U**

2. **Forgot the passcode?**
   - Check the README or this guide
   - It's NOT case-sensitive but spelled: P-H-O-E-N-I-X

3. **Game runs slowly?**
   - Close other tabs/programs
   - Reduce quality in graphics settings
   - Use Chrome or Firefox for best performance

4. **Mobile support?**
   - Touch left side of screen to move left
   - Touch right side to move right
   - Tap anywhere to shoot

## 🐛 If Something Goes Wrong

1. **Open Developer Console**: Press **F12**
2. **Go to Console tab** - Look for red errors
3. **Check Network tab** - See if files loaded
4. **Refresh page**: **Ctrl+Shift+R** (hard refresh)
5. **Clear cache**: Settings → Clear browsing data
6. **Try different browser**: Chrome, Firefox, Safari, Edge

## 🎯 Gameplay Tips

### Combat:
- Lead your shots ahead of moving targets
- Use height advantage to dodge enemy fire
- Manage ammo - you have limited bullets
- Flying in loops can help evade

### Strategy:
- Stay high for better overview
- Don't let bots corner you
- Circle around obstacles
- Keep moving to avoid sustained fire

### High Scores:
- Fast, precise aiming = more kills
- Wave survival = more points
- Destroy bots quickly = boss waves
