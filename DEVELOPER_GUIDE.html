🎮 Game Mechanics

Patient Control & Movement

Movement Speed:    0.3 units per frame
Rotation Speed:    0.005 radians per pixel (mouse)
Acceleration:      0.95x per frame (drag)
Gravity:           -0.008 units per frame
Max Altitude:      100 units
Min Altitude:      1 unit

Combat System

Player Damage:     25 HP per shot
Bot Damage:        5 HP per shot
Bot Health:        100 HP
Player Health:     100 HP
Points per Kill:   100
Ammo Pool:         120 rounds

Bot AI Parameters

Detection Range:   30 units
Chase Speed:       0.01 units per frame (50% of human)
Shoot Cooldown:    30 frames (500ms at 60fps)
Patrol Waypoints:  10 randomly generated
Memory Timeout:    5000ms (forget player after 5s)

🔧 Configuration Guide

Change Passcode

File: js/passcode.js
Line: 6
Change: this.correctPasscode = 'PHOENIX';

Adjust Bot Difficulty

File: js/bots.js
- Bot Count: Line 85 (this.botCount = 5;)
- Bot Speed: Line 28 (this.speed = 0.01;)
- Shoot Frequency: Line ~150 (this.shootCooldown = 30;)

Game Physics

File: js/game.js
- Move Speed: Line 293 (const moveSpeed = 0.3;)
- Gravity: Line 460 (this.velocity.y -= 0.008;)
- Max Speed: Line 450 (const maxSpeed = 0.5;)

Visual Customization

File: css/style.css
- HUD Color: Search ".hud-left" (change #00ff00)
- Theme Colors: Line 1-50 (gradients and colors)
- Health Bar Colors: Line 185-195
